URL Shortener description

User stories

  1. I can pass a URL as a parameter via POST method and I will receive a shortened URL in the JSON response. For example {"short_url":3,"original_url":"https://nanourl.glitch.me/"}
  2. If I pass an invalid URL that doesn't follow the valid http://www.example.com format, the JSON response will contain an error instead.For example {"error":"invalid URL: goldsfsdfpi/shorturl/1"}
  3. When I visit that shortened URL, for example https://nanourl.glitch.me/api/shorturl/1 it will redirect me to my original link.
